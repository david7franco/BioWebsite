About Me

Hi, I’m David — a software engineer passionate about building automation systems, APIs, and integration pipelines that make data flow seamlessly between platforms. My background spans Python scripting, SQL database management, and backend development using modern frameworks.

I started programming out of curiosity and the desire to make repetitive work easier. Over the years, that curiosity turned into a professional skill set focused on creating tools that connect data, automate workflows, and support scalable systems. I’ve worked with NetSuite integrations, API-driven ETL processes, and full-stack web applications.

Outside of work, I enjoy exploring tech communities, experimenting with open-source projects, and writing about technical challenges I’ve solved along the way. My philosophy: keep systems simple, maintainable, and transparent.
